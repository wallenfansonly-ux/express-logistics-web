# Express Logistics

A global logistics and courier platform built with React, Vite, and Firebase.

## Features

- Real-time shipment tracking
- Multi-language support (14+ languages)
- Admin dashboard for shipment management
- Responsive design
- Modern enterprise UI

## Tech Stack

- React 18
- Vite
- React Router v6
- Firebase (Auth, Firestore, Storage)
- React Icons
- Modern CSS

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Deployment

The project is configured for Vercel deployment. Simply connect your repository to Vercel.

## Project Structure

```
src/
  components/     # Reusable UI components
  layouts/        # Page layouts
  pages/          # Route pages
  services/       # API and service functions
  firebase/       # Firebase configuration
  hooks/          # Custom React hooks
  context/        # React context providers
  utils/          # Utility functions
  assets/         # Static assets
  styles/         # Global styles
```
